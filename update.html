Changes in v0.7.1

Bug fix with CoA features

Fixed a bug introduced in v0.7 with the Change of Authorization functionality that prevented it from executing.

Changes in v0.7.0

Better handling of large amounts of groups

When large numbers of groups are present in ISE, the extension would have a very large list that required scrolling. To help with this, when there are more than 40 groups, the groups will be divided into an alphabetical submenu based on the first letter.

Now supports any number of identity groups

Previously, the API call defaulted to only pull 20 identity groups. Now, we will retrieve all groups.

Groups menu is now sorted alphabetically

The groups were somewhat randomly placed in the menu. Now we sort the groups alphabetically for a more expected way to locate the group in the menu.

Changes in v0.6.0

Update code to use Fetch API

This is only a change to the code-base to use the Fetch API instead of the XMLHTTPRequest for API calls.

Changes in v0.5.0

Support for CoA actions

Support has been added to allow for CoA actions to be called from a right-click menu for selected text. Session and CoA functionality are performed using the MNT API which is part of the MNT persona. So, please go to the ISE Assistant settings on the toolbar and ensure that the ISE MNT Node is defined to make use of this feature!

The supported CoA types are: session rerun, port bounce, and port shutdown. I make the assumption that you know what you're doing when you click one of these; however, I may add in a confirmation for the shutdown type to prevent accidental port shutdown when intending to click a different item.

Intro/Help page

Admittedly, this probably should've existed already. But, it didn't. It does now! You will also start receiving updates like this for updates. You can access it using the help page in the settings page. You can also check it out here.

Changes in v0.4.0

Support for bulk endpoint move

You can now move endpoints by selecting multiple MAC addresses. The text inbetween is irrelevant! The extension will sniff out anything that is a MAC address format for you. You could highlight a whole page of HTML if you want! This is particularly useful when highlighting multiple rows in Splunk, Cisco ISE, etc.
